    Fonts

    Built-in Fonts~

    The ESP8266 firmware only has 1 built-in font: Unscii with font size 8pt.

    The ESP32 additionally contains the Roboto Condensed font in these font sizes: 12, 16, 22 and 28pt.

    The built-in fonts can be set by using the pointsize as parameter:

    for example: 

    1
2
    p4b1.text_font=16
p4b2.value_font=8

    To use an icon in a text you need to prefix the UTF-8 value with \u. To ensure proper decoding the payload should be used with a json or jsonl command.

    jsonl example: 

    1
    {"obj":"label","id":10,"x":0,"y":0,"w":70,"h":50,"parentid":5,"text":"\uf00c OK"}

    json example: 

    1
    ["p3b10.text=\uf00c OK"]

    Custom Fonts~

    You can add a custom font by uploading it to the internal flash. Apply it as the default font on the Configuration > HASP Settings page.
